# Foundry Bulk Exporter

Foundry VTT v13 module — right-click any folder or compendium pack in the sidebar to download its documents as a ZIP archive, mirroring the folder hierarchy. Each document is serialized as JSON (same format as Foundry's built-in "Export Data").

Works with any game system.

## Features

- **Folder export** — right-click any world folder (Actors, Items, Journals, etc.) → "Exportar pasta (ZIP)" / "Export folder (ZIP)"
- **Compendium pack export** — right-click any compendium pack row → "Exportar compêndio (ZIP)" / "Export compendium (ZIP)"
- Subfolder hierarchy mirrored as ZIP subdirectories
- Native Save As dialog (Electron-compatible)

## Installation

Install via Foundry's Add-on Modules manager using the manifest URL, or clone this repo and create a junction into your `Data/modules/` directory.
